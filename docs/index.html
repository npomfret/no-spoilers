No Spoilers

The F1 race weekend schedule for your Mac — with a hard guarantee that results never appear.

Install with Homebrew
brew install --cask npomfret/tap/no-spoilers
macOS Sequoia 15+  ·  Get Homebrew  ·  Tap Uninstall
brew uninstall --cask npomfret/tap/no-spoilers

Features

🏎

Full weekend schedule

Every session from FP1 to the race — with local start times and a live countdown to the next one.

🔕

Genuinely spoiler-proof

Results are not hidden — they don't exist in the data model at all. There's no code path that could ever surface one.

📍

Always in your menu bar

Click the icon to see the current race weekend. No context switching, no app to launch.

📱

Home screen widgets

Small, medium, and large widgets show the current weekend at a glance directly from your home screen.

⚡️

Native and lightweight

Built in Swift with only Apple frameworks. No Electron, no background daemons, no telemetry.

📲

iOS app coming soon

The same schedule, the same spoiler-free guarantee — on your iPhone.
